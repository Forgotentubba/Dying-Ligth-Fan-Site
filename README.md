## 🛠️ Tecnologias

- **HTML5** — estrutura semântica das páginas
- **CSS3** — animações, flexbox, media queries e variáveis CSS
- **JavaScript** — interatividade (comentários)

---

## ✨ Funcionalidades

- ✅ Header fixo com navegação responsiva
- ✅ Banner imersivo com gradiente de transição
- ✅ Player de vídeo com trailer do jogo
- ✅ Galeria de fotos com efeito de zoom
- ✅ Cards de galeria com expansão ao hover
- ✅ Layout responsivo para mobile
- ✅ Paleta dark consistente com tema vermelho

---

## 📱 Responsividade

O site se adapta para telas menores que **768px**:
- Navegação principal ocultada, substituída pelo menu no footer
- Layout de colunas convertido para coluna única
- Imagens e vídeo ajustados para 100% de largura
